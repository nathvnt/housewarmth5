PrivacyQuest

This is a CLI security tool for performing OSINT research and discovering
personal information that data-brokers are publishing. It utilizes Puppeteer JS to
crawl and scrape various data-brokers sites, collecting information based on given input.

This tool is designed to be used for helping individuals
remove information they do not want data-brokers profiting from.
